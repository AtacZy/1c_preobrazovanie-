# Сверка накладных аптек с данными СБИС

Скрипт для сверки накладных из аптек с данными счетов-фактур из СБИС.

## Что делает

1. Загружает CSV из `Входящие.zip` (СБИС) или папки 'Входящие'
2. Загружает CSV из `Аптеки.zip` (папка `Аптеки/csv/correct/`) или папки 'Аптеки'
3. Ищет счёт-фактуру для каждой накладной
4. Сравнивает даты, ставит метку `Не совпадает!` при расхождении
5. Сохраняет результат в `Результат/ГГГГ-ММ-ДД/`

## Особенности

- Для поставщика **ЕАПТЕКА** к номеру накладной добавляется `/15`
- Учитываются типы документов: `СчФктр`, `УпдДоп`, `УпдСчфДоп`, `ЭДОНакл`
- Если в СБИС несколько записей с одним номером — берётся первая
- Дата счёта-фактуры берётся из второй колонки "Дата"
